To create a more polished and unified layout on the homepage, we're going to make two key changes:

Ensure the Guest Selector, Date Picker, and Villa Offers all share the same maximum width and alignment.

Apply a consistent background style to all three sections, matching the one used for the "Choose Your Dates" panel.

This will create a clean, cohesive main content column that looks professional on all devices.

Step 1: Update the HTML Structure
First, we need to wrap the three components in a new div called .main-content-column. This allows us to control their width and alignment from a single parent.

File: public/index.html

Before:

HTML

<div class="container">
    <header class="header">...</header>

    <div class="guest-selection">...</div>
    <div class="calendar-section">...</div>
    <div class="content">...</div>

    <footer class="footer">...</footer>
</div>
After (with the new wrapper):

HTML

<div class="container">
    <header class="header">...</header>

    <div class="main-content-column">
        <div class="guest-selection">...</div>
        <div class="calendar-section">...</div>
        <div class="content">...</div>
    </div>

    <footer class="footer">...</footer>
</div>
Step 2: Update the CSS
Next, we'll add styles for the new wrapper and update the existing component styles in public/styles.css.

File: public/styles.css

Add the new .main-content-column styles. This class will now control the width and centering.

CSS

.main-content-column {
    max-width: 800px;
    margin: 0 auto; /* Centers the column */
    width: 100%;
}
Update the component styles to unify their backgrounds and remove old width/margin rules. We will apply the linear-gradient from the .calendar-section to all three components.

Find and modify the following CSS rules:

CSS

/* BEFORE */

AFTER (Consolidated and Unified):
